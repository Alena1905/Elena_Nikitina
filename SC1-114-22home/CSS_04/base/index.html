Форматы файлов

Doc или doc (от аглийского Document) расширение имени файлов, представляющих текст, с разметкой или без. расширение doc часто использовалось для обозначения простых текстовых файлов без формирования, однако позже стало использоваться для двоичных форматов с разметкой.
Document MS Word

Portable Document Format (PDF )- межплатформенный формат электронных документов, разработанный Adobe Sistems c использованием ряда возможностьей языка Postscript.
Document PDF

Zip популярный формат сжатия данных и архивации файлов. Файл в этом формате имеет расширение zip и хранит в сжатом или несжатом виде один или несколько файлов, которые можно из него извлечь путем распоковки. Архив
